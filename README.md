  # intro-kotlin-compose
Ejercicios para empezar a practicar Kotlin con el gestor de interfaces de Android Compose.

Realiza los siguientes ejercicios, el código de la plantilla puede ser de ayuda como punto de partida para estos desarrollos. 

Crea un repositorio de github con los tres ejercicios y con las explicaciones correspondientes.

## Ejercicio 1
Crea una aplicación que muestre un **botón** con el texto *OK* y un texto en pantalla que diga “Pulsa el botón”.
Al pulsar el botón, su texto debe cambiar a *Hola mundo*.
Si se vuelve a pulsar, el botón debe regresar a su estado original (*OK*).

## Ejercicio 2
Crea una aplicación con un **campo de texto** (TextField) donde el usuario introduzca un número.
La aplicación debe comprobar si el número introducido es primo.

Debe haber un **botón Comprobar**. Al pulsarlo, la aplicación indicará si el número es primo o no.
Si el valor introducido no es un número válido, se debe mostrar un mensaje de error en color rojo.

## Ejercicio 3
La aplicación hace lo siguiente

- Un **texto** en el que dice *introduce un numero*.
- Un **textField** en el que se debe introducir el número.
  - El número que entra tiene que validarse cómo número.
  - Si no es un número debe saltar un error.
  - Gestiona el **error con Toast**.
- Un **botón añadir número** (una vez añadido el número se borra).
- Un **botón calcular promedio**: Al pulsar el botón calcular promedio calcula la media de los números introducidos.
- Una **ventana** para mostrar el resultado de la media.
- La ventana tiene que tener un **botón para reiniciar** programa. Al reiniciar, se vuelve a la ventana principal.
